addOnClose

addOnClose: function(referenceObject,
callback)

Accepts a callback that will be called when this item closes.  The referenceObject is used to facilitate removal if necessary.

removeOnClose

removeOnClose: function(referenceObject)

Removes the close event callback associated with referenceObject.

zoomIn

zoomIn: function()

Allows you to select the tab and zoom in on it, thereby bringing you to the tab in Firefox to interact with.

zoomOut

zoomOut: function(complete)

Handles the zoom down animation after returning to TabCandy.  It is expected that this routine will be called from the chrome thread (in response to Tabs.onFocus()).

Parameters

completea function to call after the zoom down animation

setZoomPrep

setZoomPrep: function(value)

Either go into or return from (depending on <value>) “zoom prep” mode, where the tab fills a large portion of the screen in anticipation of the zoom out animation.
